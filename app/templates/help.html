Glimpser Help

Getting Started

Glimpser is a powerful tool for monitoring and capturing screenshots from various sources. Here's how to get started:

  1. Add a new template by clicking on "Add Template" on the home page.
  2. Fill in the required details such as name, URL, and frequency.
  3. Click "Submit" to create your new template.

Adding a Template: Example with time.gov

Let's walk through the process of adding a template for time.gov:

  1. Click on "Add Template" on the home page to open the template form.
  2. Fill in the following details:
    • Template Name: Enter "US Official Time"
    • URL: Enter "https://time.gov"
    • Frequency (minutes): Enter "60" to capture once per hour
    • Timeout (seconds): Enter "10" for a reasonable load time
    • Notes: You can enter "Official US time website" or leave it blank
    • Object Filter: Leave blank as we don't need to filter specific objects
    • Object Confidence: Leave blank as we're not using object detection
    • Popup XPath: Leave blank as time.gov doesn't have popups we need to handle
    • Dedicated XPath: Enter "//div[@id='time-display']" to focus on the time display
    • Callback URL: Leave blank unless you have a specific callback service
    • Proxy: Leave blank unless you need to use a proxy
    • Groups: You can enter "Time" or leave it blank
    • Rollback Frames: Enter "0" as we don't need to rollback for this static display
    • Invert: Leave unchecked
    • Dark Mode: You can check this if you prefer a dark background
    • Headless: Leave checked for background processing
    • Stealth: Leave unchecked as time.gov doesn't require stealth mode
  3. Click "Submit" to create your new template.
  4. You should now see the "US Official Time" template on your home page, which will update every hour with the current official US time.

Managing Templates

You can manage your templates from the home page:

  • View all templates in a grid layout.
  • Click on a template to view its details and recent captures.
  • Edit a template by clicking "Edit Template" on its details page.
  • Delete a template using the "Delete Template" button on its details page.

Viewing Captures

Glimpser provides several ways to view your captures:

  • On the home page, you can see the most recent capture for each template.
  • Click on a template to view its recent screenshots and videos.
  • Use the "Live" page to see real-time updates of your captures.
  • The "Captions" page shows AI-generated descriptions of your captures.

Offline Access

This help page is available offline once you've visited it while online. You can access it anytime, even without an internet connection.
