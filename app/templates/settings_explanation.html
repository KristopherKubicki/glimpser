Settings Explanation

Setting Explanation
API_KEY
Generated by the platform

This key is automatically generated by the platform and cannot be changed through the settings interface. To change it, you must manually run the generate_credentials function.
CHATGPT_KEY
Required for ChatGPT functionality

This key is necessary for the application to use ChatGPT features. The app will not perform any ChatGPT-related tasks unless this key is set.
DATABASE_PATH
Path to the database file

This setting specifies the location of the database file. The default path is 'data/glimpser.db'.
PORT
Default port for the application

The default port on which the application runs is 8082.
USER_NAME
Name of the user

This setting specifies the name of the user for the application.
MAX_WORKERS
Maximum number of concurrent threads

This setting determines the number of threads the system should run. It should be set close to the maximum number of devices you want to poll simultaneously.
LLM_CAPTION_PROMPT
Prompt for image captioning

This prompt instructs the Language Model on how to generate captions for images and their associated notes.
LLM_SUMMARY_PROMPT
Prompt for summarizing captions

This prompt guides the Language Model on how to summarize the aggregate of recently seen captions.
MAX_RAW_DATA_SIZE
Maximum size for raw data

This setting specifies the maximum size (in bytes) for raw data storage.
MAX_IMAGE_RETENTION_AGE
Maximum retention age for images

This setting determines how long (in days) images are retained before being deleted.
MAX_VIDEO_RETENTION_AGE
Maximum retention age for videos

This setting determines how long (in days) videos are retained before being deleted.
MAX_IN_PROCESS_VIDEO_SIZE
Maximum size for in-process videos

This setting specifies the maximum size (in bytes) for videos that are currently being processed.
